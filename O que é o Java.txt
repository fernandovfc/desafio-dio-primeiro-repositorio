Tipos de dados

No Java existem algumas palavras reservadas para a representação dos tipos de dados básicos que precisam ser manipulados para a construção de programas. Primitive Types: 

- int
- byte
- short
- long
- float
- double
- boolean
- char

Não são considerados objetos. Representam valores brutos.

Declaração de variáveis

Variável é uma identificação de um espaço em memória utiizado pelo nosso programa.

tipo de dados + identificação + valor atribuído

<tipo> <nomeVariavel> <atribuicaoDeValorOpcional>

ex: 

int idade

int anoFabricacao

souble salarioMinimo
